# SafariYYDS-js

A simple `npx` edition inspired by [Lakr233/SafariYYDS](https://github.com/Lakr233/SafariYYDS).

## Usage

```bash
npx safariyyds
```

Optional flags:

- `--json`: Print machine-readable JSON.
- `--help`: Show help.

## Development

```bash
bun install
bun run check
bun run build
bun run dev -- --json
```
